# AI Website Copy Generator for Local Businesses

## Business Chosen
Bean & Brew Café, Hyderabad

## Tools Used
- ChatGPT
- Google Docs
- GitHub

## Objective
To create a prompt system that generates website copy for a local café.

## What I Built
- Homepage copy
- Services page content
- CTA sections

## Prompt Logic
The prompts were designed to:
- generate a strong homepage headline
- write service descriptions
- create persuasive CTA sections
- keep the tone friendly and simple

## Output
The generated copy was made for a café website and is ready to publish.
