# FUTURE_PE_01

# AI Website Copy Generator for Local Businesses

## Business Chosen
**Bean & Brew Café**
Hyderabad, Telangana

## Project Overview
This project was created as part of Future Interns Prompt Engineering Task 1.

The objective was to design a structured prompt system that generates conversion-focused website copy for local businesses. The generated content includes homepage copy, services page content, and persuasive call-to-action sections.

## Tools Used
- ChatGPT
- GitHub
- Lovable

## Prompt Framework
The prompt system was designed to:

- Generate homepage copy with a clear value proposition
- Create service descriptions for a café business
- Write persuasive CTA sections
- Maintain a friendly and welcoming tone
- Produce website-ready content

## Prompt Logic

The prompts were structured to:
1. Define the business type and location.
2. Specify the target audience.
3. Set the desired tone and style.
4. Generate website-ready content sections.
5. Ensure persuasive and customer-focused messaging.
